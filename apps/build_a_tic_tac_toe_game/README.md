# Tic Tac Toe Game

A simple command-line Tic Tac Toe game built in Python.

## Features
- **Game Board**: In progress (basic board rendering implemented)

## How to Run
1. Ensure you have Python 3.x installed.
2. Clone this repository.
3. Run the game using:
   ```bash
   python src/main.py
   ```

## Recent Changes
- Initial project setup with `BoardPrinter` for rendering the game board.
- Added test harness for future development.
