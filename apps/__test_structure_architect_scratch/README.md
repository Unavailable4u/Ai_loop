# Tic Tac Toe Game

A simple command-line Tic Tac Toe game built in Python. Players take turns marking spaces on a 3x3 grid, aiming to get three of their marks in a row horizontally, vertically, or diagonally.

## Features

### ✅ Done
- **GameBoardModel**: Core game board logic and state management
- **Validator**: Input validation for moves and game rules
- **RequestParser**: Parses and processes player input
- **UIUpdater**: Handles display updates and user interface rendering
- **AddTask**: Task management for game workflow

### 🚧 In Progress
- **Add Task**: Additional task-related functionality (details pending)

## How to Run

1. Ensure you have Python 3.8+ installed.
2. Clone the repository or download the source files.
3. Run the game:
   ```bash
   python src/main.py
   ```
